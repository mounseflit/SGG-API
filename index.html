Bulletin Officiel API

A powerful, easy-to-use RESTful API for accessing and analyzing Moroccan Bulletin Officiel documents in both French and Arabic languages

Try Live Demo

API Overview

The Bulletin Officiel API provides comprehensive programmatic access to Morocco's official governmental publications, enabling developers to retrieve, search, and analyze legal documents, decrees, and announcements published by the Moroccan government. This RESTful API offers both metadata and full-text content from the latest bulletins in Arabic and French languages.

Getting Started

Everything you need to start using the Bulletin Officiel API

Base URL

All API requests should be made to the following base URL:

https://bo-ma-api.vercel.app

Make sure to append the specific endpoint path to this base URL when making requests.

Authentication

Currently, the API is available for public use without authentication requirements. However, rate limiting may apply to prevent abuse.

Example Request:

curl https://bo-ma-api.vercel.app/api/BO/FR

French Bulletin API

Access the latest French language publications of the Bulletin Officiel

French Bulletin Metadata French

Retrieve comprehensive metadata about the most recent French Bulletin Officiel publication.

GET /api/BO/FR

This endpoint returns key metadata including the bulletin ID, number, publication date, and direct URL.

Example Response:

{
  "BoId": 5789,
  "BoNum": "7214",
  "BoDate": "2023-06-22T00:00:00.000Z",
  "BoUrl": "https://www.sgg.gov.ma/Portals/0/BO/2023/BO_7214_Fr.pdf"
}
Try this endpoint

French Bulletin Full Text French

Extract the complete textual content from the latest French Bulletin Officiel.

GET /api/BO/Text/FR

This endpoint performs advanced text extraction from the PDF document, making the content available for search, analysis, or integration into your applications.

Example Response:

{
  "text": "ROYAUME DU MAROC BULLETIN OFFICIEL Édition de traduction officielle...[truncated]"
}
Try this endpoint
